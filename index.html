Weather App

Built with HTML, CSS and JavaScript

This weather app was orignially built using Vanilla JavaScript and Bootstrap and transferred to the React framework. It makes use of the API at openweathermap.org in order to display the local and predicted weather for a chosen city. There is also a capability to select the weather forecast for your current location.
